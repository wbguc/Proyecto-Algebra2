# Proyecto 2: Analisis de la Red Facebook WOSN Wall

Este repositorio contiene la implementación de un PageRank sobre una base de datos que muestra como se conectan los usuarios de la red social Facebook a traves de sus publicaciones.

## Estructura principal del Proyecto:
- ``main.ipynb``: Jupyter Notebook principal con todo el preprocesamiento, analisis y visualizacion.
- ``data\raw``: Carpeta que contiene la base de datos descargada
- ``Proyecto_2.pdf``: Informe sobre el proyecto.

## Recomendaciones para el correcto ejecutamiento de ``main.ipynb``:
- Clonar el repositorio
- Ejecutar todas las celdas de codigo de ``main.ipynb``
